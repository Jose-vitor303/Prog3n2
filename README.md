# Sobre o projeto 

A aplicação consiste em um sistema de biblioteca, o gerenciador desse sistema é capaz de adicionar livros, leitor, emprestimos e autores.

# Tecnologias Utilizadas 

- JavaFX
- PostgreSQL
- JUnit
- Hibernate 
- Maven
